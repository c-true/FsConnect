    Carrier Operations

    Simulation Variable Description Units Type Settable
    BLAST SHIELD POSITION Indexed from 1, 100 is fully deployed, 0flat on deck Percent_over_100 TYPE_FLOAT64
    CABLE CAUGHT BY TAILHOOK A number 1 through 4 for the cable number caught by the tailhook. Cable 1 is the one closest to the stern of the carrier. A value of 0 indicates no cable was caught. Number TYPE_BOOL
    CATAPULT STROKE POSITION Catapults are indexed from 1. This value will be 0 before the catapult fires, and then up to 100 as the aircraft is propelled down the catapult. The aircraft may takeoff before the value reaches 100 (depending on the aircraft weight, power applied, and other factors), in which case this value will not be further updated. This value could be used to drive a bogie animation. Number TYPE_FLOAT64
    HOLDBACK BAR INSTALLED Holdback bars allow build up of thrust before takeoff from a catapult, and are installed by the deck crew of an aircraft carrier. Bool TYPE_BOOL
    LAUNCHBAR HELD EXTENDED This will be True if the launchbar is fully extended, and can be used, for example, to change the color of an instrument light. Bool TYPE_BOOL
    LAUNCHBAR POSITION Installed on aircraft before takeoff from a carrier catapult. Note that gear cannot retract with this extended. 100 = fully extended. Refer to the document Notes on Aircraft Systems. Percent_over_100 TYPE_FLOAT64
    LAUNCHBAR SWITCH If this is set to True the launch bar switch has been engaged. Bool TYPE_BOOL
    NUMBER OF CATAPULTS Maximum of 4. A model can contain more than 4 catapults, but only the first four will be read and recognized by the simulation. Number TYPE_UINT32
    SURFACE RELATIVE GROUND SPEED The speed of the aircraft relative to the speed of the first surface directly underneath it. Use this to retrieve, for example, an aircraft's taxiing speed while it is moving on a moving carrier. It also applies to airborne aircraft, for example when a helicopter is successfully hovering above a moving ship, this value should be zero. The returned value will be the same as GROUND VELOCITY if the first surface beneath it is not moving. Feet_per_second  
    TAILHOOK HANDLE True if the tailhook handle is engaged. Bool TYPE_BOOL
